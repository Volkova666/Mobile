Для полного запуска программного модуля следует:
1. Поместить входной файл data.csv и lab2.exe в одну директорию.
2. Запустить lab2.exe.
3. Сначала ввести дату, за которую небоходимо получить график.
4. Вывод работы программы будет осуществлён в окно WindowsForms.

Исходный код находится в файле Form1.cs.
